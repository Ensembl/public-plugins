Species trees

The Compara pipelines use two main species trees.

  1. The NCBI taxonomy (i.e. topology only) in:
    • the Protein-trees pipeline
      • for all divisions except Vertebrates.
    • the CAFE pipelines (Gene Gain/Loss trees), with branch lengths coming from the TimeTree database
  2. A tree with branch-lengths computed in-house (available for download here) in:
    • the Protein-trees and ncRNA-trees pipelines in the Vertebrates division.
    • the Vertebrates Multiple-alignment pipelines
    • the Vertebrates Multiple-alignment pipelines
    • the Vertebrates Constrained Elements / Conservation Scores pipelines
    Briefly, the tree is computed as follows:
    1. The species-tree pipeline begins with a lightweight annotation of the input genomes. This is achieved by aligning a BUSCO [1] protein set to the genomic sequences using miniprot [2] (version 0.12-r237). From these alignments, the corresponding coding sequences (CDS) are extracted.
    2. Next, the CDS for each gene are aligned across all species to generate codon alignments. This multiple sequence alignment is primarily performed using PAGAN [3] (version v.0.61), with PRANK [4] (version v.170427) used as a fallback option if the former cannot be calculated.
    3. From the translations of these individual codon alignments, a phylogenetic tree is inferred for each gene using IQ-TREE [5] (2.2.0.3 COVID-edition). These individual gene trees are then reconciled by Astral [6] (version 5.7.1) to generate a consensus species tree.
    4. This consensus species tree is then used as a starting topology for a refinement process. A supermatrix is created by concatenating the protein alignments of all genes and it is trimmed by using TrimAl [7] (version v1.4.rev15). Then IQ-TREE is used to performs a new, full tree search on this supermatrix to find the optimal species tree topology, using the Astral tree as the starting topology. This step refines both the topology and branch lengths of the final tree.
    5. Finally, branch lengths for this definitive topology are also computed from the fourfold degenerate sites of the concatenated codon alignment (used in Cactus [8] multiple genome alignments).
