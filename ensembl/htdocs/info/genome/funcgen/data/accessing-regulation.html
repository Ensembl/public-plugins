Accessing Ensembl Regulatory Annotation

Regulation data in Ensembl can be accessed through:

  • The Ensembl Browser (see below)
  • Ensembl FTP
  • BioMart (Ensembl Regulation database).

Browser access

Regulation data can be accessed in the browser in various ways:

Location Tab

You can configure the Region in detail panel to display tracks linked to regulation. The Regulation tracks are grouped into subdivisions:

  • Regulatory Features: these can be visualised using the 'Regulatory features' track.
  • Activity by Cell/Tissue: select the tissue or cell type (epigenome) of interest to display peaks, signal and regulatory feature activity across the genome.
  • Other regulatory regions: imported tracks from external databases.

Regulation tab

Clicking on a regulatory feature will open a Regulation tab with information about the evidence supporting that regulatory feature as well as cell-specific activity estimates. Different views can be selected:

  • Activity: this view displays the selected regulatory feature and a summary of its regulatory activity across cell types.
  • Source Data: this view displays the list of source data for the chosen regulatory feature.
