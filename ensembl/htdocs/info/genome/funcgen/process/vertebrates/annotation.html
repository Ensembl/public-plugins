Ensembl Regulatory Annotation

The results of peak calling are used to identify potential regulatory regions in the genome, which include Promoters and Open Chromatin regions.

This initial release of regulatory features for a number of farmed animal species results from work supporting the GENE-SWiTCH and AQUA-FAANG consortia.

Overview of the annotation process

The steps used for defining the set of Regulatory Features (RFs) are:

  1. Retrieve ATAC-seq peaks for all the epigenomes of the target species, restricted to the peaks of canonical chromosomes.
  2. Merge peaks across epigenomes to obtain a set of unique peaks (UPs).
  3. Identify UPs overlapping with Transcription Start Sites (TSSs) and merge those within 50 bp of each other. Those that don't exceed 1.5 kb in length are classified as candidate Promoters.
  4. Group the remaining UPs (longer than 1.5 kb or not overlapping a TSS) and merge those within 50 bp of each other. These are candidate Open Chromatin regions.
  5. Some Open Chromatin regions might now overlap a TSS and not exceed 1.5 kb in length. These are also classified as Promoters.
  6. These steps result in a set of regulatory features that are classified as either Promoter or Open Chromatin.

Activity levels for each epigenome are then calculated by overlapping the defined regulatory features with the ATAC-seq peaks of each epigenome.
