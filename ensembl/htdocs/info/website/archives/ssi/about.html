About Archive Ensembl

The main Ensembl site www.ensembl.org is updated with the latest data on a monthly basis.

The Ensembl Archive sites have been set up so there is a stable link for three years to pages from a particular release (e.g. version 56 from September 2009) in Ensembl.

These links will be suitable for referring to Ensembl pages in publications.

A few older archives may be maintained if they include significant datasets for key species, but the availability of these sites cannot be guaranteed.

As from August 2014 we are providing a version of Ensembl supporting the GRCh37 human assembly. Unlike the usual Ensembl archive sites, this will be maintained and updated with new features with each release of Ensembl.

For all enquiries, please contact the Ensembl HelpDesk.
